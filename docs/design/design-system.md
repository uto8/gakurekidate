## 2. タイポグラフィ

### 2-1. フォントファミリー

| 用途 | フォント | 理由 |
|---|---|---|
| 見出し・ロゴ | **Noto Serif JP** | 知性・品格・和の洗練 |
| 本文・UI テキスト | **Noto Sans JP** | 高い可読性・モダン |

読み込み方法: Google Fonts（`display=swap`）

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@400;500&family=Noto+Serif+JP:wght@300;400&display=swap" rel="stylesheet">
```

## 5. シャドウ

現時点では未定。ダークテーマのため影よりも `bg-elevated` による積み重ね（elevation）で奥行きを表現する方針。

| トークン | 値 | 用途 |
|---|---|---|
| `shadow-sm` | 未定 | カードの浮き出し（必要であれば） |
| `shadow-md` | 未定 | モーダル・ドロワー |
| `shadow-lg` | 未定 | マッチバナー |

> **暫定方針:** `box-shadow` ではなく `backdrop-filter: blur()` + 背景色変化でレイヤーを表現する。シャドウが必要と判断した時点で具体値を追加する。

## 6. ブレークポイント

| 名称 | 幅 | 対象デバイス | 主な変化 |
|---|---|---|---|
| `sm`（モバイル） | `< 480px` | スマートフォン（375px〜430px 基準）| ベースデザイン。カードグリッド 2列 |
| `md`（タブレット） | `480px〜` | 大型スマートフォン・小型タブレット | カードグリッド 3列 |
| `lg`（デスクトップ） | 未定 | タブレット横・PC | 未定（MVP はモバイル Web が主対象）|

```css
/* Tailwind CSS v4 のカスタムブレークポイント例 */
@custom-media --md (min-width: 480px);
@custom-media --lg (min-width: 未定);
```

> **方針:** MVP はモバイル Web ファースト。タブレット・デスクトップのレイアウト調整は MVP 後フェーズで検討。`lg` ブレークポイントの具体値は未定。
