## 2. ブランドアイデンティティ

### アプリ名
**学歴デート**

### ロゴ（ワードマーク）

| 項目 | 仕様 |
|---|---|
| スタイル | ワードマークのみ（アイコンなし） |
| フォント | Noto Serif JP — Light (300) |
| カラー | ゴールド `#C9A84C`（ダーク背景上） |
| サイズ | ナビ・ヘッダー: 20px / スプラッシュ: 32px |
| カーニング | `letter-spacing: 0.12em`（高級感のため広めに） |

**ワードマークの使用ルール:**
- 必ずダーク背景の上に置く（白背景には使わない）
- ゴールドグラデーション版は LPのヒーローなど特別な場面のみ
- アプリ内ヘッダーでは `学歴デート` の文字のみ（余計な装飾なし）

## 4. タイポグラフィ

### フォントファミリー

| 用途 | フォント | 理由 |
|---|---|---|
| ロゴ・見出し | **Noto Serif JP** | 知性・品格・和の洗練 |
| 本文・UI テキスト | **Noto Sans JP** | 高い可読性・モダン |

### フォントスケール（モバイル基準）

| 用途 | サイズ | ウェイト | フォント |
|---|---|---|---|
| ページタイトル（H1） | 22px | 400 | Noto Serif JP |
| セクション見出し（H2） | 18px | 400 | Noto Serif JP |
| カード見出し（H3） | 16px | 500 | Noto Sans JP |
| 本文 | 15px | 400 | Noto Sans JP |
| ラベル・バッジ | 13px | 500 | Noto Sans JP |
| キャプション・注釈 | 12px | 400 | Noto Sans JP |

### テキストルール
- 行間（line-height）: 本文は `1.7`、見出しは `1.3`
- 文字間隔（letter-spacing）: 見出し `0.04em`、ロゴ `0.12em`、本文 `0`
- 長文の最大幅: `640px`（モバイルでは `100%`）

## 6. コンポーネント仕様

### ボタン

**Primary（メインアクション）**
- 背景: ゴールドグラデーション `accent-gradient`
- テキスト: `#0A0A0A`（黒）、Noto Sans JP Medium 15px
- 高さ: `52px`
- 幅: 基本フルワイド
- 状態: ホバー→明度 +10%、押下→明度 -10%、disabled→`opacity: 0.4`

**Secondary（サブアクション）**
- 背景: `transparent`
- ボーダー: `1px solid #C9A84C`
- テキスト: `#C9A84C`、15px
- 高さ: `52px`

**Ghost（破壊的アクション・退会など）**
- 背景: `transparent`
- テキスト: `#888888`、14px
- ボーダーなし

### 入力欄（Input / Textarea）

- 背景: `#1E1E1E`
- ボーダー: `1px solid #2A2A2A`
- フォーカス時ボーダー: `1px solid #C9A84C`
- テキスト: `#F0F0F0`、15px
- プレースホルダー: `#555555`
- 高さ（Input）: `52px`
- パディング: `0 16px`
- エラー時ボーダー: `1px solid #E05252` + エラーテキストを下に表示

### プロフィールカード（探索一覧）

```
┌────────────────────┐
│                    │
│   写真 (4:5)       │  ← border-radius: 12px
│                    │
│ 名前 24歳          │  ← F0F0F0 / 16px Medium
│ 〇〇大学 △△学部   │  ← C9A84C / 13px（ゴールドで学歴を強調）
│ 自己紹介テキスト   │  ← 888888 / 13px / 2行まで
└────────────────────┘
```

- 背景: `#141414`
- グリッド: 2列固定（モバイル基準）/ 480px 以上は 3列
- カード間隔: `8px`
- 写真なし時: `#1E1E1E` のグレープレースホルダー（イニシャル中央表示）
- **いいねボタンはカードに置かない**（詳細画面のみ）。カードタップで詳細へ遷移する

### 学歴バッジ（EducationBadge）

- 背景: `transparent`
- テキスト: `#C9A84C`（ゴールド）、13px
- 先頭に `🎓`（任意）または `◆` 記号
- 他の情報と視覚的に区別するため必ずゴールドを使う

### いいねボタン（LikeButton）

**配置: プロフィール詳細画面の最下部に固定（一覧カードには置かない）**

- 未いいね: Primary ボタン（ゴールドグラデーション）＋ハートアイコン + 「いいね」テキスト
- いいね済み: Secondary ボタン（ゴールド枠）＋ハート塗りつぶし + 「いいね済み」テキスト、タップ不可
- 送信中: `opacity: 0.5`、pointer-events none
- タップ時アニメーション: scale（`1 → 1.03 → 1`、200ms）

### マッチングバナー（MatchBanner）

- 全画面オーバーレイ: `rgba(0,0,0,0.85)` + `backdrop-filter: blur(4px)`
- 中央カード背景: `#141414`
- 上部装飾: ゴールドグラデーションの細いライン（`3px`）
- テキスト: 「マッチング成立！」を Noto Serif JP 22px でゴールド表示
- 2人のアバターを横並び・中央揃え
- CTA: 「メッセージを送る」Primary ボタン + 「あとで」Ghost テキスト

### チャット画面

| 要素 | 仕様 |
|---|---|
| 背景 | `#0A0A0A` |
| 自分のバブル | 背景 `#C9A84C`（ゴールド）、テキスト `#0A0A0A`、右寄せ |
| 相手のバブル | 背景 `#1E1E1E`、テキスト `#F0F0F0`、左寄せ |
| 退会済みバブル | 背景 `#1A1A1A`、テキスト `#555555`、「退会済みユーザー」ラベル付き |
| バブル角丸 | `12px`（送信者側の底角のみ `4px`） |
| 時刻 | `#555555`、12px、バブル下 |
| 入力エリア | `#141414`、top border `#2A2A2A` |

### ボトムナビゲーション

- 背景: `#0A0A0A` + top `1px solid #2A2A2A`
- 4タブ: 探す / いいね / マッチング / プロフィール
- 非アクティブアイコン: `#555555`
- アクティブアイコン: `#C9A84C`（ゴールド）
- ラベル: 12px、アクティブ時のみゴールド表示

## 7. アニメーション・トランジション

| シーン | 種類 | 時間 |
|---|---|---|
| ページ遷移 | fade（`opacity: 0→1`） | 200ms ease |
| カードタップ | scale（`1→0.97`） | 100ms ease |
| いいねボタン | scale（`1→1.2→1`） | 200ms spring |
| マッチバナー表示 | fade + scale（`0.95→1`） | 300ms ease-out |
| ボトムシート展開 | slide-up | 300ms ease-out |
| エラーメッセージ | fade-in | 150ms ease |

**方針:** アニメーションは控えめに。動きを「見せる」より「感じさせる」程度に留める。

## 8. 画面別デザイン方針

### ログイン / 登録画面
- 背景: `#0A0A0A` 全面
- 上部に `学歴デート` ロゴ（ゴールド、Noto Serif JP）を中央配置
- フォームはページ中央に縦積み
- 「ログイン」ボタン = Primary（ゴールド）

### オンボーディング画面
- ステップインジケーター（細いゴールドのプログレスバー）を上部に表示
- 1ステップ = 1スクロール画面に収める
- 学歴入力ステップは最後に配置し「完了すると探索が始まります」と明示

### 探索画面（/discover）
- ヘッダー: `学歴デート`（左）+ フィルターアイコン（右、ゴールド）
- フィルターパネル: ボトムシートで展開
- カードグリッド: 2列、縦スクロール
- 各カードで学歴をゴールドで強調表示

### プロフィール詳細画面
- 写真を画面上部フルワイドに配置（アスペクト比 4:5）
- 写真の下にグラデーションオーバーレイ → 名前・年齢を写真上に白テキストで重ねる
- 学歴セクションにはゴールドの左ボーダーラインを付ける（強調）
- 「いいね」ボタンは画面下部に固定表示（Primary ボタン）

### マッチング一覧画面
- アバター（円形） + 名前 + 学歴 + 最終メッセージの行リスト形式
- 未読バッジ: `#C9A84C`（ゴールド小円）

### チャット画面
- ヘッダー: 相手の名前（左）+ 大学名（小テキスト）
- スクロール方向: 下が最新
- 入力エリアは画面最下部に固定

---

## 9. アクセシビリティ配慮

- ゴールド（`#C9A84C`）と黒（`#0A0A0A`）のコントラスト比: **約 7.1:1**（WCAG AA 合格）
- タップターゲット最小サイズ: **44×44px**
- 全フォームにラベルを付ける（`aria-label` または `<label>`）
- プロフィール写真に `alt` テキストを設定（名前を使用）
- アニメーションは `prefers-reduced-motion` を考慮して無効化できる設計
